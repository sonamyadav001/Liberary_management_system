SHABD SANGRAH 📚
Smart Library Management System
Shabd Sangrah is a modern web-based platform designed to automate library operations. It bridges the gap between traditional reading and digital convenience by integrating AI assistance and a smart delivery system.
🚀 Key Features
 * 🤖 AI Chatbot: Handles real-time queries regarding book availability and fine status.
 * 📱 QR Integration: Unique QR codes for seamless book issuing and returning.
 * 🚚 Smart Delivery: A service-oriented module for book delivery within a 5km radius (₹5/km).
 * 📊 Live Dashboard: Interactive UI with real-time analytics, Dark Mode, and a physical clock.
 * 📧 Automation: Automated fine calculation (₹5/day) and email notification system.
🛠️ Tech Stack
 * Backend: Python (Flask)
 * Frontend: HTML5, CSS3, JavaScript
 * Database: SQL (SQLite)
 * Hosting: Render
